# mocha-chai-test

mocha-chai-test to verify charge differential

# Setup the project

create a repo
cd repo
npm init
npm i mocha --save-dev
npm i chai --save-dev

created src to write the functions
created test to write the test

# To run the test

# npm run test

## To generate mochawesom report

## npm run generate:report
